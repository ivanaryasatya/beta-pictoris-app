About The Project

The Beta Pictoris platform represents a paradigm shift in autonomous robotics. Designed for high-precision tactile operations, it integrates advanced telemetry with robust kinetic capabilities.

Equipped with omni-directional mobility and a modular payload system, it serves as the ultimate testbed for next-generation remote operation protocols.

Key Features

📡

Multi-Protocol Comms

Supports WiFi, ESP-NOW, Bluetooth, and LoRa for robust long-range control.

📷

FPV Camera System

Real-time video feed with custom overlays for immersive first-person operation.

🧭

IMU Stabilization

6-axis gyro and accelerometer integration providing precise orientation data.

🤖

Autonomous Mode

Programmable sequence engine for executing complex maneuvers automatically.

🎯

Precision Shooters

Dual-motor flywheel mechanism with adjustable angle and power control.

📊

Live Telemetry

Real-time charts for battery voltage, motor temperature, and signal strength.

How It Works

1. Connect

Power on the robot and connect via the Dashboard using your preferred protocol (WiFi/ESP-NOW).

2. Control

Use keyboard WASD keys or on-screen controls to navigate. Adjust shooter angle and speed.

3. Engage

Lock onto targets using the camera feed and fire projectiles with precision.
